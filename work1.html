📋 Задание №1: Простой AI-помощник

Цель: Научиться подключаться к API нейросети, отправлять вопросы и получать ответы.

Что делаем: Создаём чат-бота, который отвечает на вопросы пользователя через модель Llama 3.3 (бесплатно через Groq).

Результат: Рабочий AI-бот на вашей странице.

🤖 AI-помощник

Задайте любой вопрос — я постараюсь ответить понятно!

✏️ Напишите вопрос и нажмите "Отправить"...

🔹 Ключ можно получить на console.groq.com (регистрация → API Keys)

📖 Что означает каждая строчка кода?

🔹 fetch('https://api.groq.com/openai/v1/chat/completions')
Отправляет HTTP-запрос к серверу Groq. Это как "позвонить" нейросети и задать вопрос.
🔹 method: 'POST'
Тип запроса. POST значит "я отправляю данные на сервер" (в отличие от GET — "дай мне данные").
🔹 headers: { 'Authorization': `Bearer ${apiKey}` }
Передаём API-ключ. Это как пропуск — сервер проверяет, что вы зарегистрированы и можете пользоваться сервисом.
🔹 body: JSON.stringify({ model: 'llama-3.3-70b-versatile', messages: [...] })
Тело запроса. Внутри указываем: какую модель использовать (llama-3.3), какие сообщения передаём (системная инструкция + вопрос пользователя).
🔹 temperature: 0.7
"Творчество" AI. От 0 до 1. Чем выше, тем более неожиданные и креативные ответы. 0.7 — золотая середина.
🔹 max_tokens: 150
Максимальная длина ответа в "токенах" (кусочках слов). 150 токенов ≈ 100-120 слов.
🔹 async / await
Специальные слова для работы с запросами. "async" говорит: "функция будет ждать ответа". "await" говорит: "подожди здесь, пока придёт ответ".
🔹 try { ... } catch (error) { ... }
Ловим ошибки. Если что-то пошло не так (нет интернета, неверный ключ), программа не упадёт, а покажет понятное сообщение.
