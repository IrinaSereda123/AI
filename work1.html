🎯 Задание

Создать веб-страницу с AI-помощником, который:

  1. 🤖 принимает вопрос от пользователя;
  2. 📡 отправляет запрос к Groq API (бесплатная языковая модель);
  3. 💬 получает ответ и показывает его на странице;
  4. 🎨 имеет красивый и понятный интерфейс.

📖 Часть 1. Что такое API?

API (Application Programming Interface) — это способ, с помощью которого разные программы могут общаться друг с другом. Представь, что ты в ресторане: ты (твоя программа) делаешь заказ официанту (API), а он приносит его с кухни (сервер).

🔍 Как это работает в нашей задаче:
  • Твоя страница отправляет вопрос к Groq API.
  • Groq API передаёт вопрос нейросети (модели Llama 3.3).
  • Нейросеть генерирует ответ и возвращает его через API.
  • Твоя страница получает ответ и показывает его пользователю.

🌐 Что такое Groq?

Groq — это сервис, который предоставляет бесплатный доступ к мощным языковым моделям (как ChatGPT). У них есть API, к которому мы можем обращаться из JavaScript.

Для работы нам понадобится API-ключ — это как пароль, который даёт доступ к сервису.

🚀 Шаг 3. JavaScript — подключение к API

✏️ Твоя задача:

Создай файл script.js и скопируй этот код. Вместо ВСТАВЬ_СВОЙ_КЛЮЧ нужно будет вставить свой API-ключ, который ты получишь на сайте Groq.

🔑 Как получить API-ключ (бесплатно):
1. Перейди на https://console.groq.com.
2. Зарегистрируйся (можно через Google или GitHub).
3. Перейди в раздел «API Keys» → «Create API Key».
4. Скопируй ключ и вставь его в код вместо ВСТАВЬ_СВОЙ_КЛЮЧ.
📄 script.js — логика работы с API
1// --- Ждём, пока загрузится страница --- 2document.addEventListener('DOMContentLoaded', function() { 3 // --- Находим элементы на странице --- 4 const button = document.getElementById('askBtn'); 5 const questionInput = document.getElementById('questionInput'); 6 const answerDiv = document.getElementById('answerResult'); 7 8 // --- ОЧИСТКА ПЛЕЙСХОЛДЕРА ПРИ КЛИКЕ --- 9 const defaultQuestion = "Что такое искусственный интеллект?"; 10 11 questionInput.addEventListener('focus', function() { 12 if (questionInput.value === defaultQuestion) { 13 questionInput.value = ''; 14 questionInput.style.color = '#333'; 15 } 16 }); 17 18 questionInput.addEventListener('blur', function() { 19 if (questionInput.value === '') { 20 questionInput.value = defaultQuestion; 21 questionInput.style.color = '#aaa'; 22 } 23 }); 24 25 // --- ВСТАВЬ СЮДА СВОЙ API-КЛЮЧ (получи на groq.com) --- 26 const apiKey = 'ВСТАВЬ_СВОЙ_КЛЮЧ_СЮДА'; 27 28 // --- Системный промпт (как AI должен себя вести) --- 29 const systemPrompt = `Ты дружелюбный AI-помощник для детей. 30 Отвечай кратко (2-3 предложения), понятно и с юмором. 31 Используй простые слова. Если вопрос сложный, объясняй на примерах.`; 32 33 // --- Функция для отправки вопроса --- 34 async function askAI() { 35 let question = questionInput.value.trim(); 36 37 if (question === defaultQuestion || question === '') { 38 answerDiv.innerHTML = '<p class="hint">Пожалуйста, введи свой вопрос!</p>'; 39 return; 40 } 41 42 answerDiv.innerHTML = '<p class="loading">🤔 AI думает...</p>'; 43 44 try { 45 const response = await fetch('https://api.groq.com/openai/v1/chat/completions', { 46 method: 'POST', 47 headers: { 48 'Authorization': `Bearer ${apiKey}`, 49 'Content-Type': 'application/json' 50 }, 51 body: JSON.stringify({ 52 model: 'llama-3.3-70b-versatile', 53 messages: [ 54 { role: 'system', content: systemPrompt }, 55 { role: 'user', content: question } 56 ], 57 temperature: 0.7, 58 max_tokens: 150 59 }) 60 }); 61 62 if (!response.ok) { 63 const errorData = await response.json(); 64 throw new Error(errorData.error?.message || `Ошибка ${response.status}`); 65 } 66 67 const data = await response.json(); 68 const answer = data.choices[0].message.content; 69 70 answerDiv.innerHTML = ` 71 <div class="answer-text"> 72 <strong>🤖 AI:</strong> ${answer} 73 </div> 74 <small style="color: #999; margin-top: 10px; display: block;"> 75 Ответ сгенерирован моделью Llama 3.3 76 </small> 77 `; 78 79 } catch (error) { 80 console.error('Ошибка:', error); 81 answerDiv.innerHTML = ` 82 <p class="error">❌ Ошибка: ${error.message}</p> 83 <p style="color: #666; font-size: 14px;"> 84 Проверь API-ключ или подключение к интернету 85 </p> 86 `; 87 } 88 } 89 90 button.addEventListener('click', askAI); 91 questionInput.addEventListener('keypress', function(e) { 92 if (e.key === 'Enter') askAI(); 93 }); 94});
📖 Что нужно знать для понимания кода:
• async/await — специальный синтаксис для работы с запросами.
• fetch() — функция для отправки HTTP-запросов.
• try/catch — ловит ошибки, чтобы программа не падала.
• Bearer ${apiKey} — способ передачи ключа доступа к API.

📖 Подробный разбор кода (по строчкам)

📌 Строка 2: document.addEventListener('DOMContentLoaded', ...)
Ждём, пока вся страница загрузится, и только потом запускаем код. Без этого скрипт может не найти элементы на странице.
📌 Строки 4-6: const ... = document.getElementById(...)
Находим элементы на странице по их id: кнопку, поле ввода, блок для ответа.
📌 Строки 11-23: Очистка плейсхолдера
Когда пользователь кликает на поле ввода — стандартный текст "Что такое искусственный интеллект?" исчезает. Если поле пустое и пользователь ушёл — текст возвращается.
📌 Строка 26: const apiKey = 'ВСТАВЬ_СВОЙ_КЛЮЧ_СЮДА'
Сюда нужно вставить API-ключ, полученный на console.groq.com. Это как пароль для доступа к нейросети.
📌 Строки 29-31: const systemPrompt = `...`
Системный промпт — это инструкция для AI. Здесь мы говорим модели, как себя вести: "Ты дружелюбный помощник для детей, отвечай кратко". Можно менять как угодно.
📌 Строка 34: async function askAI()
async — говорит, что функция будет работать с запросами, которые могут выполняться долго (например, ждать ответ от сервера).
📌 Строка 35: let question = questionInput.value.trim()
Берём текст из поля ввода и удаляем лишние пробелы в начале и конце (.trim()).
📌 Строка 44: try { ... } catch (error) { ... }
Блок try/catch ловит ошибки. Если внутри try что-то пошло не так (нет интернета, неверный ключ), программа не упадёт, а выполнит код в catch.
📌 Строка 45: fetch('https://api.groq.com/...')
fetch() — отправляет запрос к серверу. Это как "позвонить" нейросети. URL указывает, куда звонить (адрес API Groq).
📌 Строка 46: method: 'POST'
Тип запроса. POST значит "я отправляю данные на сервер". Есть ещё GET ("дай мне данные"), но для чата с AI нужен POST.
📌 Строка 48: 'Authorization': `Bearer ${apiKey}`
Передаём API-ключ. Это как пропуск — сервер проверяет, что вы зарегистрированы. Bearer — стандартный способ передачи ключа.
📌 Строка 51-59: body: JSON.stringify({ ... })
Тело запроса. JSON.stringify превращает объект в строку для отправки.
  📌 model: 'llama-3.3-70b-versatile'
Какая модель AI будет отвечать. Llama 3.3 70B — мощная бесплатная модель от Meta.
  📌 messages: [ ... ]
Список сообщений. Каждое сообщение имеет role (кто говорит) и content (текст). role: 'system' — инструкция, role: 'user' — вопрос пользователя.
  📌 temperature: 0.7
"Творчество" AI. От 0 до 1. Чем выше, тем более неожиданные и креативные ответы. 0.7 — золотая середина. 0 — сухие факты, 1 — фантазии.
  📌 max_tokens: 150
Максимальная длина ответа в токенах. Токен — это кусочек слова. 150 токенов ≈ 100-120 слов. Если нужно длиннее — увеличьте число.
📌 Строка 67: const data = await response.json()
await — ждём, пока сервер ответит. .json() — превращает ответ сервера из текста в объект JavaScript.
📌 Строка 68: const answer = data.choices[0].message.content
Достаём текст ответа AI из полученного объекта. choices[0] — первый вариант ответа (их может быть несколько).
📌 Строки 90-93: button.addEventListener('click', askAI)
Вешаем обработчик: когда пользователь нажимает на кнопку или клавишу Enter — вызывается функция askAI.
